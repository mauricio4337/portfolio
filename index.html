World Weather Analysis

In this project I an API call to get current weather data. I used matplot lib to analyze the weather data. Then, I used Google MAPS and Places API to create Maps with markers of vacation locations based on weather criteria. Finally, I created routes between nearby hotels in cities that fit the vacation weather criteria. Technologies and Languages used for this project: Python, Google APIs, Matplotlib.

Mapping Earthquakes

In this project I refined my JavaScript skills and continued improve my use of CDNs to enhance a website. I used Leaflet and Mapbox to display GeoJson data as markers, circle markers, lines, polygons on map layer tiles. The final part of the project collects earthquake data from the past week and gives a dynamic mapping of the recent earthquakes world wide.
